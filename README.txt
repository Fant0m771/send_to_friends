Drupal Send to Friends module:
----------------------
Overview:
--------
This module provides a block allowing users send
interesting page addresses from your website to friends.

Installation:
------------
1. Copy the send_to_friends module to the Drupal sites/all/modules/
   or sites/example.com/modules/ directory.
2. Go to "administer" -> "modules" and enable the module.
3. Check the "use send to friends form" -> "access control" page to
   enable use of this module to different roles.
4. Set up your email messages and fields count in
   "administer" -> "configuration" -> "Send to Friends".
5. Enable the "Send to friends form" under "site building" -> "blocks".
